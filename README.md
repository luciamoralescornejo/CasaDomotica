El asistente por voz espera la llamada "Paco" y luego espera un comando. 
Los comandos por voz se basan en la union de dos palabras, una accion y un elemento.

Acciones: pon, quita, enciende, apaga, abre, cierra

Elementos: television, aire, luz, temporizador, puerta

La television tiene varios estados como noticias, tiempo, YouTube. A estos estados los llamamos por ejemplo con: pon + notcias.

El aire y el temporizador tambien tienen sus funcion especial de asignarles un numero, a esta se le llama con pon + aire + numero, hay que hacer una breve pausa para decir el numero
